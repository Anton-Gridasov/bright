Profiler

Profiler is a great tool for making code profiling and call time measurements.

Usage

Profile could be called by Bright singleton:

br()->profiler()

Or by creating instance of BrProfiler:

require_once('path/to/your/bright/BrProfiler.php');
$profiler = new BrProfiler();

Methods

start($name)

Put call to start() before the code you are going to measure

br()->profiler()->start('operation1');
// some time consuming code

You can do nested calls to start():

br()->profiler()->start('operation1');
// some time consuming code
br()->profiler()->start('operation2');
// another one time consuming code

finish($name)

Use finish() to get the time in microseconds since call to regarding start() command

$microsconds = br()->profiler()->finish('operation1');
echo('Operation1 took ' . $microseconds . ' msecs'));

logStart($name)

Same as start() but writing writing operation name into log file

logStart($name)

Same as finish() but with writing operation name and duration into log file
